# JavaScript and React Concepts

## Array vs Set
Array allows duplicate values and maintains order. Set stores only unique values and removes duplicates automatically.

## Object vs Map
Object stores key value pairs with string keys. Map allows keys of any type and provides better performance for frequent additions and deletions.

## Array Methods
Common methods include push to add elements, pop to remove last element, includes to check existence, and many functional methods.

## ES6
ES6 introduced features like arrow functions, destructuring, template literals, let and const, and modules.

## let vs var vs const
var is function scoped. let is block scoped. const is block scoped and cannot be reassigned.

## DOM
DOM represents the structure of a web page. It allows JavaScript to access and modify HTML elements.

## Synchronous and Asynchronous
Synchronous code executes line by line. Asynchronous code allows tasks to run later without blocking execution.

## Arrow Function
Arrow functions provide a shorter syntax for writing functions and do not have their own this binding.

## map filter reduce
map transforms each element. filter selects elements based on condition. reduce accumulates values into a single result.

## Storage
Local storage stores data in the browser with no expiration. It is used for saving user data on client side.

## Cookies
Cookies store small pieces of data in the browser and are sent with every HTTP request.

## useState
useState is a React hook used to manage state in functional components.

## useEffect
useEffect is used to handle side effects like API calls and lifecycle events.

## useContext
useContext is used to share data across components without passing props manually.

## useReducer
useReducer is used for complex state management using reducer functions.

## Debounce
Debounce delays execution of a function until a certain time has passed since the last call.

## Throttling
Throttling limits how often a function can be executed within a given time interval.
