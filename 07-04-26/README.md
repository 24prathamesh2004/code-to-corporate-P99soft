# Day 2 Learning

## File Handling
File handling is used to read and write data to files. Common modes include read, write, and append.

## Types of Databases
SQL databases store structured data in tables. NoSQL databases store flexible data such as JSON. ORM allows developers to interact with databases using objects instead of writing SQL queries. GraphQL is a query language that allows clients to request only the required data.

## SQLAlchemy
SQLAlchemy is a Python ORM used to define database tables as classes and interact with databases efficiently without writing raw SQL queries.

## API Concepts
Headers contain metadata such as authentication tokens and content type. Status codes indicate the result of an API request such as 200 for success, 201 for creation, 400 for bad request, 401 for unauthorized, 404 for not found, and 500 for server error. URLs define API endpoints. Dependency injection is used to provide reusable components like database connections or authentication logic.

## Constructor and Destructor
Constructor is automatically called when an object is created and is used for initialization. Destructor is called when an object is destroyed and is used for cleanup.

## Decorators
Decorators are used to modify the behavior of functions without changing their actual code by wrapping them with additional functionality.

## Async and Await
Async and await are used for asynchronous programming. They allow non blocking execution of tasks, especially useful for IO operations.

## ASGI vs WSGI
WSGI is used for synchronous Python applications. ASGI supports asynchronous programming and is used in modern frameworks like FastAPI.

## Git
Git is a version control system used to track changes in code. Common commands include init, add, commit, and push.

## Postman
Postman is a tool used to test APIs by sending HTTP requests and analyzing responses.
